# Family Nutrition App

Web app HTML/CSS/JavaScript đơn giản để quản lý dinh dưỡng cho gia đình.

## Tính năng

- Quản lý danh sách thành viên gia đình
- Nhập bữa ăn hàng ngày theo từng thành viên
- Tính tổng calo, protein, carb và fat trong ngày
- Theo dõi tiến độ calo theo mục tiêu mỗi người
- Hiển thị biểu đồ calo theo tuần
- Lưu dữ liệu trong trình duyệt bằng `localStorage`

## Cách chạy

Mở file `index.html` trực tiếp bằng trình duyệt.
